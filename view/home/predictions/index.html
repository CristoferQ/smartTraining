Data Set

As many datasets as sectors are considered to possess the protein, each of these is trained with different algorithms and applying various performance measures.

Predictors for vPHL

The predictors for pVHL were obtained by training with Random Forest, using an iteration of 100 trees, the global efficiency reaches values above 97% and there is no adjustment because it is trained by leave one out.

Algorithm Application

The prediction algorithm can be used to assess the clinical relevance of a point-in-time mutuation. For this, the position, the protein sector and the original and mutated amino acid must be selected, obtaining the clinical relevance as a result.

Full data set v/s Data set by sector

When considering the complete set of data sets, the results obtained when training the predictors turned out to be poor, obtaining values of performance measures of 0.5 p.m., which is why the division in the specific sectors of the pVHL protein was considered.

When training the predictors with different data sets associated with specific sectors of the protein, the performance increased considerably, achieving an average of 0.97.

To corroborate that the performance results were correct, random subdivisions of the data set were made n times, obtaining average values of performance of 0.6 which denote that the subdivision was correct and had biological justification.
