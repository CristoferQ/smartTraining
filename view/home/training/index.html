MLSMPc for training models

Meta Learning System for Mutations Protein classificator is the methodology used for VHL-Hunter to training and selected models. MLSMPc use meta learning for increase the performance and create classificators more accurate and precise.

Using interaction protein-protein sector

MLSMPc needs to split the data set to create models associated with these groups. Protein-protein interaction sectors were used to achieve this. 13 meta classificator was created using this methodology associated to each superfice interaction sector.

Splitter Validation

To validate the partitions induced by the protein-protein interaction sectors, divisions were induced using clustering techniques, recursive clustering and random partition distributions, in order to corroborate the validity of the addition of topological information.

How to work MLSMPc

Input data and splitter criteion

The data set includes phylogenetic and structural information, adding topological information of the protein referring to the protein-protein interaction sectors to induce the partitions. Each partition is a new dataset.

Training and selected models

Each partition represents a set of data to train. The models are trained using an exploratory phase of supervised learning algorithms. The selection of the models is generated by statistics associated with the performance obtained.

Meta Learning System and result classification

All model selected are used for implements a meta learning system. it create a probability distribution associated to result of classification and evaluated the error of result in a function of distribution of accuracy for each model.
